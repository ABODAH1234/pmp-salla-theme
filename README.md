# PMP Performance — Salla Twilight Theme

الثيم الرسمي المخصص لمتجر **Power Match Performance** على منصة سلة.

## الهوية والتجربة

- تصميم عربي RTL بهوية سوداء وحمراء وبيضاء.
- واجهة مخصصة لقطع الأداء والمحركات.
- مسارات تصفح لمحركات HEMI وLS/LT وCoyote وGodzilla و1FZ و2JZ.
- أقسام مخصصة لأنواع القطع والعلامات التجارية.
- بنرات PMP الأصلية محفوظة داخل المستودع، بلا اعتماد على روابط خارجية.
- دعم صفحات المنتج، البحث، الحساب، السلة، الدفع، المدونة وخدمة العملاء.
- متوافق مع محرك Salla Twilight.

## التشغيل المحلي

يتطلب Node.js 22.18 أو أحدث وpnpm.

```bash
pnpm install
pnpm production
```

تُكتب ملفات الإنتاج في مجلد `public/`.

## الاستيراد إلى سلة

المستودع جاهز للاستيراد مباشرة من فرع `main`. ملف تعريف الثيم `twilight.json` موجود في جذر المستودع كما تتطلب سلة.

من بوابة شركاء سلة: **الثيمات → إنشاء/استيراد ثيم → GitHub**، ثم اختر:

```text
ABODAH1234/pmp-salla-theme
```

بعد الاستيراد، اربط نسخة تجريبية من المتجر واختبر المعاينة قبل تفعيل الثيم كنسخة افتراضية.
